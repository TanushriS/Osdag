## ✨ Overview

The **Osdag Bridge Deck Designer** empowers structural engineers and students to interactively design bridge decks. With real-time visual feedback, users can modify parameters and immediately see the impact on the bridge's cross-section and plan view.

## 🚀 Key Features

*   **🎨 Real-time Visualization**: Dynamic SVG rendering that adapts instantly to your inputs.
*   **📏 Unit Versatility**: Seamlessly toggle between **Metric (m)** and **Imperial (ft)** units.
*   **💾 Smart Auto-Save**: Never lose your work—progress is saved locally automatically.
*   **👁️ Accessibility First**: Includes a **High Contrast** mode for improved visibility.
*   **📱 Responsive Design**: Fully responsive UI built with `shadcn/ui` and `Tailwind CSS`.

## 🏁 Getting Started

### Prerequisites

*   Node.js (v18+)
*   npm (v9+)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/irajverma/Osdag-Bridge-Deck-Designer.git
    cd Osdag-Bridge-Deck-Designer
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the dev server**
    ```bash
    npm run dev
    ```

## 📦 Build & Deploy

**Production Build**:
```bash
npm run build
```

**Deploy to Firebase**:
```bash
firebase deploy --only hosting
```
